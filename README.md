# AI-Powered Fake News Detector for Students

## Description
This project detects whether a news article is REAL or FAKE using Google's Gemini AI.

## Technologies Used
- Python
- Streamlit
- Gemini 2.5 Flash
- Generative AI

## Features
- Fake news detection
- Confidence score
- AI explanation
- Student awareness tips

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
